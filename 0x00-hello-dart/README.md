Web Dart - Hello Dart
=============

The Hello Dart project is an introductory to the Dart programming language.

Installation on Windows (run Command Prompt as administrator):
-----------

```
choco install dart-sdk
choco upgrade dart-sdk
```


Usage
-----

```
dart [filename]
```
